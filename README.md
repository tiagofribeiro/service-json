<br>

## Run locally

To run the development server, clone the repo, install the packages present in `package.json` and run:

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
